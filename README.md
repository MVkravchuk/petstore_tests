# How to run

```commandline
# Install dependensies
pip install -r requiremennts.txt

# Run tests without Allure report
pytest

# Run tests with Allure report
pytest --alluredir=./.my_allure_results

# To see Allrure report 
allure serve .my_allure_results    
```
